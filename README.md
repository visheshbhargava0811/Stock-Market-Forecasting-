# Stock-Market-Forecasting-
A batch ETL pipeline that turns financial news into trading signals through five stages: FinBERT sentiment scoring → triplet extraction → embedding → semantic search → LLM reasoning. It pulls ~8–15 articles per day across 8 stocks from the MarketAux API, with per-batch checkpointing so API-bound jobs resume cleanly after a crash. A validation step parses the model's JSON output and drops malformed rows before they reach backtesting. Roughly 65K knowledge-graph triplets are embedded, with dot-product similarity retrieving the top-K most relevant context per prediction. Confidence signals were backtested in Lumibot from Jan 2022 to Apr 2024 — the knowledge-graph strategy outperformed LLM reasoning, sentiment-only, and buy-and-hold on 5 of 8 stocks, including NVDA at +410% vs +239%.
